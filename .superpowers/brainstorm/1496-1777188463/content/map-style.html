A. Stamen Watercolor + 青色滤镜

实现：Stadia Maps 现成水彩瓦片，前端 CSS 套青色 hue-rotate

优：1 天能上线、可缩放、动态平移

劣：本质是西方水彩画风，不是真水墨；像"水彩涂的杭州"

B. Stadia 极简底图 + 自绘水墨叠层

实现：Stadia Alidade Smooth 当骨架，上面叠 SVG 水墨皴笔（手绘西湖、远山、飞鸟）

优：保留地图的可缩放可定位 + 真水墨味；产品上线最适合

劣：要画 SVG 元素，多花 1-2 天

C. 全静态手绘水墨图

实现：一张完整的水墨画 PNG/SVG，标记是绝对定位的 hot-spot

优：视觉最纯粹、最古风

劣：不能缩放平移、不能扩展到其他城市；以后再做"杭州外"很难复用

三个缩略图都是示意，不是最终成品。点上面的卡片选你心仪的方向，或在终端回复 A / B / C。

我的推荐是 B：因为你选了"上线、云开发、扩展性"路径，B 既保留水墨味，又能自然扩展到其他城市/景点。
