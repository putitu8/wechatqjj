# 古镇文化地图小程序 · UI/UX 设计规范

- **日期**：2026-04-26
- **配套**：`2026-04-26-wechat-su-dongpo-design.md`（产品设计文档）
- **范围**：补全产品文档已有的视觉方向，输出可直接落地的设计系统、页面布局、组件 anatomy、交互动效和验收清单
- **基调**：基于 ui-ux-pro-max 的 "E-Ink / Paper" 风格 + 中文古典装帧的混合 —— **东方书页（Eastern Paper）** 风格

---

## 0. 风格定位（Eastern Paper）

| 维度 | 选择 | 不选 | 理由 |
|---|---|---|---|
| 母型风格 | E-Ink / Paper | Glassmorphism / Neumorphism | 阅读优先、calm、可访问性 AAA |
| 中式叠加 | 双线回字纹 + 竖排标签 + 印章红 | 仿古花格满铺 | 装饰只在「框」与「红印」上，中央留白 |
| 动效底色 | 慢节奏 + spring 物理 | Motion-driven 滚动视差 | 文化场景需要 calm，不是 wow |
| 暗色策略 | **不做暗色**（首版） | 翻转配色 | 米色宣纸 = 全局核心情绪；夜间用降低亮度的同一色板代替（见 §1.6） |

**一句话**：「在白纸上轻拓墨痕，红印为眼，竖字为骨」。

## 1. 设计系统（补齐版）

### 1.1 颜色（语义化 token，含 spec 已定义）

```
/* 墨色（前景 / 描边 / 主标） */
--ink-deep:    #1F3A40   /* 主标题、深墨 */
--ink-cyan:    #2D5560   /* 副标、地图青 */
--ink-light:   #5A8A92   /* hover / 远山 */

/* 纸色（背景 / 卡片） */
--paper-rice:  #F7EEDA   /* 全局背景 */
--paper-warm:  #F5E9C8   /* 卡片 / 抽屉提升层 */
--paper-deep:  #EDDEB6   /* 输入框、禁用底色（新增） */

/* 装帧 */
--frame-brown: #5A3C1F   /* 双线回字纹 */
--frame-light: #B8966B   /* 次级边框（新增） */

/* 印章 / 状态 */
--seal-red:    #8B3A3A   /* 已打卡红印、错误 */
--seal-light:  #B86060   /* 红印 hover（新增） */

/* 文字 */
--text-main:   #3A2818   /* 正文 */
--text-fade:   #7A6A4F   /* 辅助、时间戳 */
--text-mute:   #A89880   /* placeholder、禁用文字（新增） */

/* 功能（新增） */
--success:     #4A6B3F   /* 完成/成功 */
--warn:        #B8742A   /* 距离不足等温和提示 */
```

**对比度自检（WCAG AA 4.5:1）**：
- `--text-main` on `--paper-rice` = **9.8:1** ✓
- `--text-fade` on `--paper-rice` = **4.6:1** ✓（边界，不可再浅）
- `--text-mute` on `--paper-rice` = **3.1:1** ✗ 仅用于装饰文字，不可承载语义
- `--ink-cyan` on `--paper-rice` = **6.2:1** ✓
- `--seal-red` on `--paper-rice` = **5.8:1** ✓

### 1.2 字号刻度（移动端 rpx；1rpx = 0.5px @375）

| token | rpx | 用途 |
|---|---|---|
| `--fs-xs` | 20 | 时间戳、辅助 |
| `--fs-sm` | 24 | 卡片副文 |
| `--fs-base` | 28 | 正文（保 16px 等价，避 iOS 自动缩放） |
| `--fs-md` | 32 | 卡片标题、tab 标签 |
| `--fs-lg` | 40 | 页面标题 |
| `--fs-xl` | 56 | 诗句、英雄标 |
| `--fs-poem` | 48 | 竖排诗句标签（行高 1.4） |

行高：正文 1.6；标题 1.3；竖排诗 1.4 + `letter-spacing: 8rpx`。

### 1.3 字体策略

- **标题 / 诗 / 竖排标签**：思源宋体 SemiBold（`Noto Serif SC 600`）— 子集化 ~1.2 MB，`wx.loadFontFace` 异步加载
- **正文**：系统字体回退栈 `-apple-system, "PingFang SC", "Source Han Serif SC", serif`
- **印章数字 / 经度纬度**：`SF Mono, Menlo, Consolas, monospace`（防 tabular 数字跳动）
- **回退期**：字体未加载时全部走系统字，**不阻塞渲染**（`font-display: swap` 等价）

### 1.4 间距刻度（4rpx 基底）

```
--sp-1: 8rpx    --sp-2: 16rpx   --sp-3: 24rpx
--sp-4: 32rpx   --sp-5: 48rpx   --sp-6: 64rpx
--sp-7: 96rpx   --sp-8: 128rpx
```

**节奏规则**：
- 卡片内 padding：`--sp-4`
- 卡片间 gap：`--sp-3`
- 段落间 vertical：`--sp-5`
- 页面左右安全留白：`--sp-4`（手机）；横屏 / 大屏增至 `--sp-6`

### 1.5 圆角与边框

| token | 值 | 场景 |
|---|---|---|
| `--r-none` | 0 | 古籍框、印章方章 |
| `--r-sm` | 8rpx | 输入框 |
| `--r-md` | 16rpx | 普通卡片 |
| `--r-lg` | 24rpx | 弹层、抽屉、悬浮按钮 |
| `--r-pill` | 999 | 圆形按钮、徽章 |
| `--bw-thin` | 2rpx | 单线 |
| `--bw-thick` | 4rpx | 双线回字纹外框 |
| `--bw-double-gap` | 6rpx | 双线之间留白 |

### 1.6 阴影 / 提升（5 级 elevation）

「纸」不该有强阴影，全部用浅墨晕。

```
--ele-0: none                                       /* 平铺正文 */
--ele-1: 0 2rpx 8rpx rgba(31, 58, 64, 0.06)         /* 卡片 */
--ele-2: 0 4rpx 16rpx rgba(31, 58, 64, 0.10)        /* 抽屉、悬浮按钮 */
--ele-3: 0 8rpx 32rpx rgba(31, 58, 64, 0.14)        /* 模态、证书 */
--ele-press: 0 1rpx 2rpx rgba(31, 58, 64, 0.20)     /* 按压凹陷 */
```

夜间模式（首版不做，但预留）：将 `--paper-rice` 替换为 `#2A241A`，墨色翻为米白，红印改 `#C26A6A`。

### 1.7 动效 token

| token | 时长 | 缓动 | 用途 |
|---|---|---|---|
| `--mo-fast` | 150ms | `ease-out` | 按压、ripple |
| `--mo-base` | 240ms | `cubic-bezier(.2, .8, .2, 1)` | 状态切换 |
| `--mo-page` | 320ms | `cubic-bezier(.2, .8, .2, 1)` | 页面/抽屉转场 |
| `--mo-stamp` | 480ms | spring (mass=1, stiffness=180, damping=14) | 红印盖章 |
| `--mo-typing` | 60ms/字 | linear | 东坡打字 |

**关键**：所有动效必须响应 `prefers-reduced-motion` —— 关闭后保留状态变化但去除位移/缩放，缩短至 80ms 直接切换。

### 1.8 z-index 标尺

```
0    背景纸
10   普通卡片
20   sticky 头部
40   悬浮东坡按钮
60   抽屉
80   模态 / 证书
100  toast / 错误条
1000 调试边界（dev only）
```

## 2. 通用组件 anatomy

### 2.1 `<ancient-frame>` 古籍装帧标签

**视觉解剖**（核心组件，全站复用）：

```
┌──────────────────────────┐  ← 外双线 frame-brown 4rpx
│  ┌────────────────────┐  │
│  │                    │  │  ← 内单线 frame-brown 2rpx
│  │       [text]       │  │     之间空 6rpx（form 回字纹空)
│  │      writing       │  │
│  │      vertical      │  │  竖排正文：Noto Serif SC 600
│  │                    │  │  text 颜色 --ink-deep
│  └────────────────────┘  │
└──────────────────────────┘
   四角内描 8rpx 短墨点（仿古书钉装饰，可关）
```

**props**：
| prop | 类型 | 默认 | 说明 |
|---|---|---|---|
| `text` | string | — | 标签文字（自动竖排） |
| `size` | sm/md/lg | md | sm=120×64rpx ｜md=160×96rpx ｜lg=240×160rpx |
| `clickable` | boolean | false | 是否可点（含 `:active` 按压态） |
| `corner-mark` | boolean | true | 四角墨点装饰 |
| `seal` | boolean | false | 右下角小红印（用于「已打卡」状态） |

**states**：
- `:active`：整体下沉 2rpx + ele-press
- 不可访问者：`<text aria-hidden="true">` 包裹纯装饰元素，主标用真文字（屏幕阅读器可读出）
- **横排回退**：当 `text.length > 8` 时强制横排，避免文字串成长条

### 2.2 `<floating-su>` 全局悬浮东坡

```
位置：right: 32rpx; bottom: tab-bar-height + safe-area + 50rpx
尺寸：120×120rpx 圆形（命中区扩到 144×144rpx）
内容：东坡像（圆形头像）+ 周围细环 frame-light 2rpx
气泡："问我" 24rpx，frame-brown 描线，每 8s 微抖（rotate ±3deg, 800ms spring）
```

**关键 UX 修订（spec 未覆盖）**：
- ❌ 不出现于：`/pages/chat/`、登录卡片可见时
- ✅ 拖拽：长按 600ms 进入拖拽模式，可吸附到屏幕左 / 右两侧任意位置（lstorage 持久化）
- ✅ 「微抖」首次进入页面后只触发 1 次，之后 30s 才允许下一次（避免疲劳）
- ✅ `accessibility-label="问问东坡先生"` + `accessibility-role="button"`
- ✅ 与 web-view 抽屉冲突时：抽屉打开后悬浮按钮 240ms 淡出，关闭后淡入

### 2.3 印章组件 `<seal-stamp>`

```
方章：宽高 80×80rpx，--seal-red 实底，米色刻字 "東坡"（楷体或装饰汉字）
边框：1rpx 米色内描（仿真石章留白）
盖章动画：
  入场 200ms 内 scale 1.4→1，opacity 0→1，rotate -8deg
  落定 spring 振 ±3deg 衰减 280ms
  落定瞬间触发 wx.vibrateShort({type:'medium'}) 触觉反馈
```

落点：景点卡片右下、证书 7 个站点格子、对话页东坡气泡左上。

### 2.4 抽屉（在 web-view 内实现）

```
状态：collapsed (60rpx 露出顶) → half (50vh) → full (90vh)
手势：垂直拖拽，松手吸附到最近态（spring 320ms）
顶部把手：48×4rpx 灰条 + 上下 24rpx padding（命中扩到 88rpx 高）
内容：竖排诗 + 横排景点简介 + 两个 CTA（问东坡 / 我已到此一游）
背景：paper-warm + ele-2，顶部圆角 --r-lg
关闭：下拉过半 dismiss、点遮罩 dismiss、Android 物理返回 dismiss
遮罩：rgba(31, 58, 64, 0.32) 渐入 240ms
```

## 4. 关键交互细节（spec 没写够的）

### 4.1 GPS 打卡的失败与降级

| 场景 | 表现 |
|---|---|
| 用户拒授权 | 卡片：「不知阁下身在何方，难辨是否到此」+「去设置」按钮 → `wx.openSetting` |
| 距离 ≤ 200m | 印章动画 + 触觉中等震动 + 进度+1 toast |
| 距离 > 200m | 温和提示（不报错风格）：「离 X 还差约 X 米，再走两步罢」 |
| 网络失败 | 缓存到 `checkinPending` 队列；下次进入 wusheng 时自动重试 |
| GPS 漂移误打 | （首版不做）后期可加同景点 24h 内冷却 |

### 4.2 web-view 与小程序的转场连贯性

抽屉里点「问问东坡先生」→ navigateTo chat 页：
- web-view 抽屉**不要立即关**，让转场动画结束后再关，避免视觉跳断
- chat 页入场 from-right slide 320rpx + fade（spring 320ms）

### 4.3 字体加载策略

```
1. 应用启动时 wx.loadFontFace 异步加载思源宋体子集
2. 加载期：所有标题 / 诗句先用系统字呈现（不留空 / 不闪烁）
3. 加载成功：240ms 内 cross-fade 切换到思源宋体
4. 加载失败：保留系统字，不重试不报错
5. iOS 与 Android 字体包名一致，避免分平台逻辑
```

### 4.4 输入与键盘行为

- 对话页输入框 focus 时：消息流自动上推 = 键盘高度，不被遮挡
- 输入框 `placeholder="此处问东坡"`，使用 `--text-mute` 不依赖颜色传达语义（自带前置图标 ✏ 强化）
- 输入失败（空字符）发送按钮保持禁用（opacity 0.4 + 不可点）

### 4.5 错误与空态（spec §6 的 UI 落地）

| 场景 | UI 表现 |
|---|---|
| web-view 加载失败 | 全屏插画（破墨砚 SVG）+「墨砚未润，请稍后重启」+ 重试按钮 |
| 名人心迹列表加载中 | 7 个 placeholder 卡，shimmer 0.6s 周期 |
| 进度页未登录占位 | 灰度 7 个卡片 + 中央「先留名号方可记录足迹」 |
| 对话页 AI 零命中 | 兜底回复 + 3 个建议气泡（杭州 / 美食 / 兄弟） |
| 字体未加载 | 系统字 + 不阻塞（无 UI 表征，对用户透明） |

## 5. 无障碍 / 性能验收清单

### 5.1 无障碍

- [ ] 所有可点元素 ≥ 88×88rpx (44pt)
- [ ] 相邻可点元素间距 ≥ 16rpx
- [ ] 文字对比度 ≥ 4.5:1（用上面 §1.1 表格已验过）
- [ ] 竖排诗句的 `aria-label` 写横排版，便于屏幕阅读器阅读
- [ ] 图标按钮（悬浮东坡、印章 toggle）必须有 `accessibility-label`
- [ ] 红印 + 灰度卡的「已 / 未打卡」状态不只用颜色 —— 同时有图标和文字
- [ ] `prefers-reduced-motion` 关掉所有打字、印章弹跳、抖动；保留状态切换但缩短
- [ ] 字号支持系统缩放：用 rpx + flex 不写死高度，文本可换行不截断
- [ ] 焦点可见：键盘 / 外接设备聚焦时显示 2rpx ink-cyan 描边

### 5.2 性能预算

- [ ] 首屏 LCP < 2s：地图站 web-view 预加载、骨架屏占位
- [ ] 字体异步加载，不阻塞首屏
- [ ] 14 张工笔画走云存储 CDN，IntersectionObserver 懒加载
- [ ] 动效用 transform / opacity，不动 width / height / left / top
- [ ] 列表（7 道菜、7 个景点）短不需要虚拟化；对话历史 50 条上限够用
- [ ] tap 反馈在 100ms 内可见

### 5.3 跨设备

- [ ] iOS 刘海屏：safe-area-top + safe-area-bottom 已留
- [ ] Android 手势栏：底部 CTA 上移 safe-area-bottom
- [ ] 小屏 320rpx 宽：2 列网格降级 1 列
- [ ] 横屏：左右内边距加大到 sp-6；地图 web-view 限定竖屏更稳
